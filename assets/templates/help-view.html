Viewing Crypts

Last updated: 1st June 2016
  • Using The Official Client
  • Using CURL
  • Using the Website

Once you have successfully created a crypt it can be viewed with the get command.

WARNING:Viewing a crypt does not require any authentication so be careful what meta data you expose in the description

Recommended Usage

rcrypt get -crypt=0ad1f05ccfc3a01327225d9635263152

Advanced Usage

rcrypt get -crypt=0ad1f05ccfc3a01327225d9635263152 -decrypt=false
rcrypt get -crypt=0ad1f05ccfc3a01327225d9635263152 -debug=true

Viewing a crypt is a read-only action so a challenge is not needed

curl https://ripacrypt.download/1/crypt/0ad1f05ccfc3a01327225d9635263152/

To view a Crypt in your browser simply append the CryptID to the URL https://RIPACrypt.download/crypt/

e.g. https://RIPACrypt.download/view/0ad1f05ccfc3a01327225d9635263152/
